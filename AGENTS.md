# Regras do Projeto

Siga estritamente as seguintes regras durante o desenvolvimento:

1. **Firebase**: Usar Firebase (Auth e Firestore) para autenticação e banco de dados.
2. **Ícones**: Usar `lucide-react` para todos os ícones.
3. **UI**: Usar a biblioteca de componentes `shadcn/ui`.
4. **Qualidade de Código**: Todo componente deve ser seguro e fortemente tipado com TypeScript.
